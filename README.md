# smart-village-management
This is my first Git Repository.
<br>
Author- Tanuja Jangilwad

Smart Village Development project is a responsive web-based Progressive Web App (PWA) built with HTML, CSS, and JavaScript for rural management in areas like Kolhapur, Maharashtra. It digitizes village services, government schemes, and admin tools as a B.Tech/CST academic project, with versions enhanced for PWA offline support, dark mode, and real-time features.
Key Features

    Service Management: Tracks water supply, electricity, healthcare, education, weather (via OpenWeather API for Kolhapur), and emergency contacts like ambulance (108) and police (100).

    Government Schemes Portal: Lists 15-22+ schemes (e.g., PM-KISAN, Ayushman Bharat) with descriptions, eligibility, apply links, and deadlines; includes search, filters, favorites, and voice search.

    Grievance System: Users file complaints with categories (water, electricity), photo uploads, status tracking (pending/resolved), and admin dashboard for CRUD operations using localStorage.

    Advanced UI/UX: Dark/light theme toggle, responsive design (mobile-first), PWA installable/offline-first, charts (stats), PDF/CSV export, toast notifications, MFA login/register.

    Analytics & More: Role-based dashboards (citizen/admin), real-time stats, voice commands, file uploads, and integrations like geolocation

    How to use : tnjsmartvillageportal.netlify.app
